Atomic Design: Elementos Básicos

Botones sólidos

Por defecto se configuran los elementos desde el mixin de buttons
Se configuran los estilos en el atom _buttons.scss
Para los botones sólidos solo se pasa el color de background al mixin.

Botones outline

Los botones de estilo Outline se obtienen con la clase btn-outline

Para configurar un color diferente para botones Outline se hace uso del mixin BUTTON con los siguientes parametros
- $bg = color de background.
- $color: #ffffff = color de fuente.
- $border-color: transparent = color de borde.
- $border-width: 0rem = ancho de borde.
- $border-radius: 0.3rem = radio de los bordes del botón

Mixin


.at-btn-outline {
@include button(
$bg, $color: #ffffff,
$border-color: transparent, $border-width: 0rem,
$border-radius: 0.3rem
)
}

Grilla de imagenes

Para configurar la grilla de imagenes, en el archivo grid.scss se pueden observar las clases que debemos agregar a cada elemento de la grilla para que que tome el numero de columnas segun el breakpoint; Por ejemplo si deseamos que nuestra grilla tenga 3 columnas en el breakpoint lg debemos agregar la clase: grid-12 seguida de la clase del breakpoint y el numero con el cual se completarian las 12 columnas. En este caso, que queremos que se vean 3 columnas agregamos la clase grid-lg-4. La idea de usar estos numeros es para completar un total de 12 columnas, 3*4 = 12

FlexBox

Flex Direction

Se asigna la clase d-flex para implementar contenedores flex.

Flex ofrece dos maneras para la presentación del contenido row y column. Por defecto el contenido es mostrado con el estilo row sin embargo esto puede ser modificado.

Para presentar el contenido como columnas se tiene la clase fd-column. De la misma manera fd-column-reverse presentará el contenido en columnas solo que invertirá su orden
